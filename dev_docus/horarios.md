# Sistema de Gestión de horarios

* Los profesores/expertos tienen una vista donde se muestan sus horarios (si tiene más de uno en pestañas)
* Cada profesor puede crear sus propios horarios en base a una template. 
* Cuando se crea un horario, se tiene la opción de enviar para su verificación a admin mediante una solicitud de tipo: verificar horario.
* El admin recibe una entrada en su tabla de tareas con la solicitud y un enlace que permite ver el horario. 
* Cuando un admin entra a un horario de un profesor, dispone de un botón "Validar" que al pulsarlo, finaliza el flujo de la solicitud y marca el horario como validado.
* Cuando la tarea/solicitud es validada por el admin, en el horario aparece un check en la parte superior derecha y la etiqueta validado.
